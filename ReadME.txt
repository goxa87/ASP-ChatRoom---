ПРОЕКТ МЕССЕНДЖЕРА

Реализовано:
* Авторизация, аутентификация
* БД,EntityFramework
* MVC архитектура
* Внедрение зависимостей
* REST Api
* Клиентская чать JQuery

Для запуска на вашей машине необходимо в классе  DataContext изменить строку подключения на строку подключения к вашему серверу. 

Из командной строки выполнить команду update-database.

Не реализовано:
*  Обновление общего чата в реальном времени.
* Подстановка поля Pseudonym в сообщения пришедшие через фронт.
